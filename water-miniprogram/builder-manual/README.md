# AI 时代 Builder 手册

## 这是什么

这是一套独立于具体项目的母本源文件。

当前状态：

- 七章正文 v1 已完成
- 当前工作重心是一致性修订与后续派生准备

它不是：

- 送水小程序文档
- 某个 case 的复盘
- 工具说明书

它是：

- 一本关于 AI 时代 Builder 的通用手册
- 后续学习版、工程运行时版、传播版的共同源头

---

## 为什么单独放在 `builder-manual/`

因为这套内容需要和具体项目上下文解耦。

如果把它混在项目 `docs/` 里，容易出现两个问题：

1. 被单个案例绑架，失去普适性
2. 被当前对话上下文过度影响，结构不稳

所以这里单独作为母本目录。
