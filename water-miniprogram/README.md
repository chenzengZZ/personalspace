# 送水到家小程序

一个基于微信小程序的送水服务平台，用户可以浏览水品、下单购买、查看订单状态。

## 项目结构

```
water-miniprogram/
├── assets/                  # 静态资源
│   ├── icons/              # 图标文件
│   └── images/             # 商品图片
├── pages/                   # 页面文件
│   ├── index/              # 首页（商品列表）
│   ├── detail/             # 商品详情页
│   ├── cart/               # 购物车页面
│   ├── order/              # 订单相关页面
│   │   ├── list/           # 订单列表
│   │   └── detail/         # 订单详情
│   └── user/               # 个人中心
├── app.js                   # 小程序入口文件
├── app.json                 # 小程序全局配置
├── app.wxss                 # 全局样式
├── project.config.json      # 项目配置
└── sitemap.json             # 小程序索引配置
```

## 功能特性

### C端用户功能
- 浏览水品列表，支持分类筛选和搜索
- 查看商品详情，了解价格、销量、库存
- 加入购物车，批量管理商品
- 下单购买，查看订单状态
- 个人中心管理收货地址

### B端管理功能（待开发）
- 订单管理
- 数据导出与分析
- 库存管理

## 使用说明

1. 打开微信开发者工具
2. 导入项目，选择 `water-miniprogram` 文件夹
3. 填入 AppID: `wx2aead6f3e1e9bf0d`
4. 点击编译预览

## 技术栈

- 微信小程序原生框架
- WXSS 样式
- 本地存储（模拟数据）
