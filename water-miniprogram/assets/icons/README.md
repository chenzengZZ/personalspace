# 图标资源说明

本项目需要以下图标文件，请将图标文件放置在 `assets/icons/` 目录下：

## TabBar 图标 (建议尺寸: 81x81 px)
- home.png / home-active.png - 首页图标
- cart.png / cart-active.png - 购物车图标
- order.png / order-active.png - 订单图标
- user.png / user-active.png - 我的图标

## 功能图标 (建议尺寸: 48x48 px)
- search.png - 搜索图标
- add.png - 添加图标
- check.png - 勾选图标
- delete.png - 删除图标
- arrow-right.png - 右箭头
- location.png - 定位图标
- feedback.png - 反馈图标
- about.png - 关于图标
- service.png - 客服图标
- default-avatar.png - 默认头像

## 订单状态图标 (建议尺寸: 48x48 px)
- order-pending.png - 待支付
- order-delivering.png - 配送中
- order-completed.png - 已完成
- pending.png - 待处理
- success.png - 成功
- delivering.png - 配送中

## 空状态图标 (建议尺寸: 200x200 px)
- empty.png - 空状态
- empty-cart.png - 购物车为空
- empty-order.png - 订单为空

## 商品图片 (建议尺寸: 400x400 px)
放置在 `assets/images/` 目录下：
- nongfu-big.png - 农夫山泉大桶
- nongfu-small.png - 农夫山泉小瓶
- yibao-big.png - 怡宝大桶
- baishuishan.png - 百岁山
- wahaha.png - 娃哈哈
- jingtian.png - 景田

## 图标获取建议
1. 使用阿里巴巴矢量图标库 (iconfont.cn)
2. 使用 Flaticon (flaticon.com)
3. 自行设计或委托设计
